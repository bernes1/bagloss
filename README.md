# bagloss
Sometimes you wonder if it is you or github having issues?.
now you dont have to wonder you get a push notification when github has a incident.

the source of data is coming from githubstatuspage rss

## Setup
First you wil need to set these Environment variables in either in a .env file or in docker compose

```
PUSHOVER_TOKEN=
PUSHOVER_USER_KEY=
USER_TZ=
```
